Class Dice::SimpleRoll
In: dicelib.rb
Parent: Roll

This is a simplified subclass of Roll, for handling simple dice strings, like a single die roll. This operates just like the Roll class, except that it has some methods to more easily pull totals and a tally for the roll. This is for -simple- dice rolls only, no static parts or labels. Any extra parts are ignored. For example: 

  SimpleRoll.new("1d20")
       -- or --
  SimpleRoll.new("1d6e")
